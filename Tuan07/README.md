
# Tuần 07

Thiết kế giao diện react native dùng axios gọi API, áp dụng các hook như: state, reducer, ngoài ra còn custom hook api


## Installation

Clone project

```bash
  git clone https://github.com/LeDonChung/ReactNative.git
  cd Tuan07
```

Start project with npm

```bash
  npm start
```
