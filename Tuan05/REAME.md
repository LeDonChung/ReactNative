
# Tuần 05

Thiết kế giao diện react native với React Navigation, FlatList ngoài ra còn gọi api thông qua axios


## Installation

Clone project

```bash
  git clone https://github.com/LeDonChung/ReactNative.git
  cd Tuan05
```

Start project with npm

```bash
  npm start
```
