
# Tuần 04

Thiết kế giao diện react native với React Navigation, truyền data qua các màn hình



## Installation

Clone project

```bash
  git clone https://github.com/LeDonChung/ReactNative.git
  cd Tuan04
```

Start project with npm

```bash
  npm start
```
